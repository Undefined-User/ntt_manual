# 杂项

## 身份证相关

### 检查是否有效

```
/ic_check <15/18位身份证>
```

{% hint style="success" %}
会检查地区是否有效，并给出提示。
{% endhint %}

### 生成校验位

```
/ic_18 <前17位身份证>
```

根据ISO 7064:1983.MOD 11-2，使用前十七位计算最后一位。

### 身份证生成

```
/ic_gen
```

随机生成格式正确的身份证号码，一次十条。

{% hint style="success" %}
支持指定目标地区 (省 市 乡)
{% endhint %}
